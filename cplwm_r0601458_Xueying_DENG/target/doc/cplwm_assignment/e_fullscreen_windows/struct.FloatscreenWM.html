Fields

windows: Vec<Window>

The vectors of the windows, the first one is on the master tile

full_screen_windows: Vec<(Window, Geometry, FloatOrTile, Option<Window>)>

The full screen windows,contains all full_screen windows, but only the last one is visible. when you close all full screen windows. you will go back to the previous look, when you set the first full screen window. 1st is current full screen window 2nd is it proviously geometry 3rd is its type last one is the provious focused window , before that one is set to be full screen.

focused_window: Option<Window>

The vectors of the focus window, when it is empty then there is no focused window, which only can contain 1 object at maximum.

float_windows: Vec<(Window, Geometry)>

all float windows that are under controlled

tile_windows: Vec<Window>
